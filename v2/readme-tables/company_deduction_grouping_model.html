Property Value Description Notes
id integer The id of the deduction grouping.
name string The name of the deduction grouping. writable
nonelective_cap decimal The maximum annual nonelective contribution limit for deductions in the deduction grouping. writable
company_match_percent_cap decimal The maximum percent of employee contribution the employer will match for deductions in the deduction grouping. writable
company_match_percent_of_gross_cap decimal The maximum percent of gross pay that is susceptible to employer match for deductions in the deduction grouping. writable
company_match_percent_2_cap decimal The maximum percent of employee contribution that the employer will match after the first tier is maxed for deductions in the deduction grouping. writable
company_match_percent_of_gross_2_cap decimal The maximum percent of gross pay that is susceptible to employer match after the first tier is maxed for deduction in the deduction grouping. writable
version string The version of the deduction grouping. See Versioning section.
